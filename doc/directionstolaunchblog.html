Directions to Launch a Blog

Below, you'll find the directions necessary to launch a blog on blogs.columbian.com - and to verify content and required plugins to satisfy standards.

Learn more »

Adding a Blog to blogs.columbian.com


1. Columbian admin has admin rights


2. Remove default post and comment


3. Change sample page to home
     A. Change layout to "Blog Excerpt (Summary)"


4. Change default category to news


5. Settings
     A. General
          i. Timezone -> Los Angeles
          ii. Week Starts on -> Sunday

     B. Reading
          i. Static page -> Front page = Home

     C. Discussion
          i. Uncheck first 3 options

     D. Facebook comments - 122928857737763
           i. FB ID
           ii. Number of Comments = 10
           iii. Width = 605
           iv. Uncheck Show Comment Count

     E. JSON API
          i. Confirm Core is activated


6. Appearance
     A. Widgets
          i. Main Sidebar
               a. Search
               b. Text (ad code)
               c. Recent Posts
               d. Categories
               e. Tag Cloud
               f. Archives
               g. Text (blogger button)

     B. Theme options
          i. Theme Elements (check both)
          ii. Uncheck homepage override
          iii. Custom Scripts (DFP code)

     C. Header (logo)


7. Plugins
     A. Google Analytics
          i. Manually enter UA-2785447-1
          ii. Check Show advanced settings
          iii. Check the last 5 options under Custom Variables Settings


8. SEO
     A. Titles & Metas
          i. Check Force rewrite titles

     B. Social
          i. Check OpenGraph option
     ii. Add Twitter card if applicable

     C. XML Sitemap
          i. Confirm sitemap generation is working properly


9. GO LIVE!!! Set the appropriate site category. This will list the blog on the blogs landing page.


10. Add to columbian.com listings


11. Add to Verve app
